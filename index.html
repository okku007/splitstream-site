Under the hood

For the engineers.

SplitStream is a Kotlin Multiplatform app with a single shared core. One repository owns sync, cache and every snapshot read; both platforms wrap it rather than re-implementing logic. Widgets never sync — they render the cached snapshots the app publishes.

Kotlin Multiplatform Jetpack Compose Jetpack Glance + RemoteViews SwiftUI + WidgetKit SQLDelight Ktor kotlinx.serialization kotlinx.datetime WorkManager + AlarmManager
